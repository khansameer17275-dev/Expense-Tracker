# Expense-Tracker
project overveiw

Expense Tracker is a simple web-based application built using HTML, CSS and JavaScrit.

This project helps users manage their daily income and expenses by allowing them to add transactions, veiw total balance,track income and expences separately,delete transactions, and filter records.

All data is stored using localStorage, so transactions remain saved even after refreshing the browser.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Fetures

* Add Income and Expense Transactions
* Real-time Balance Calculation
* separate Income and Expenses Summary
* Delete Transactions
* Filter by:
    * All
    * Income
    * Expense
* Data Persistence using localStorage
* Responsive and Clean UI

  -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

  Technologies Used

  * HTML5
  * CSS
  * JavaScript
  * localStorage API

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

How it Works

1. User enters:
  * Transaction Title
  * Amount
  * Type (Income/Expense)
2. On clicking Add Transaction:
  * Data is validated
  * Transaction is added to the list
  * Balance is saved on localStorage
3. User can:
  * Delete any transaction
  * Filter transactions by type
